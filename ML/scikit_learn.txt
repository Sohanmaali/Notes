--------------------------------------------------

PHASE 1 — Core sklearn workflow ✅ MOST IMPORTANT 
-----------------------------------------------
Learn this pattern:

Load data
→ train_test_split
→ preprocess
→ model
→ fit
→ predict
→ evaluate
→ transform

Key topics:
- train_test_split
- fit(), predict()
- accuracy_score, mean_squared_error
- Classification vs Regression

Practice:
- Iris classification
- California housing regression

--------------------------------------------------

PHASE 2 — CLASSIFICATION FUNDAMENTALS 
-----------------------------------------------
Models:
- LogisticRegression
- KNeighborsClassifier
- DecisionTreeClassifier
- RandomForestClassifier

Concepts:
- Precision, recall, accuracy
- Confusion matrix
- Class imbalance

Project:
- Binary classification (Titanic / churn / loan)
- Use:
  - StandardScaler
  - Pipeline
  - classification_report

--------------------------------------------------

PHASE 3 — REGRESSION FUNDAMENTALS 
---------------------------------------------
Models:
- LinearRegression
- Ridge, Lasso
- RandomForestRegressor

Metrics:
- MSE
- MAE
- R²

Project:
- House prices / sales / exam scores
- Compare:
  - Scaled vs unscaled data
  - Linear vs tree models
